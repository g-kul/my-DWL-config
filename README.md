# Configuration
- changed some key bindings (volume, brightnesscontrol using brightnessctl and window management)
- terminal [foot terminal](https://codeberg.org/dnkl/foot)
- menu - [wmenu](https://codeberg.org/adnano/wmenu/)
- lock - [hyprlock](https://github.com/hyprwm/hyprlock/)
- background wallpaper - [swaybg](https://github.com/swaywm/swaybg)
- font used - [Terminus  bitmap](http://terminus-font.sourceforge.net/) and [Terminus-nerd-font](https://github.com/ryanoasis/nerd-fonts)
- colors - tokyo night
